## Introdução - CSS Flexbox

Um dos pilares para construir páginas responsivas, certamente é o FlexBox. O FlexBox usa segmentos lógicos, usando cálculos baseados em porcentagem para ajustar elementos em um determinado espaço definido.

Algum dos comandos de FlexBox e seus significados de disposição dos elementos:

* flex-direction: direção dos elementos filhos (vertical ou horizontal);
* flex-wrap: quando elementos não cabem em linha, ele desliza para baixo;
* align-items: define o alinhamento dos elementos no eixo X;
* align-content: define como os elementos vão ser mostrados na tela;
* flex-flow: junta os comandos flex-direction e flex-wrap.
